## Installation

Clone the repository and run `bun install`

```bash
bun install
```

## Usage

Running with Development Mode

```bash
bun run dev
```

Building with Production Mode

```bash
bun run build
```

Running with Production Mode

```bash
bun run start
```

Migrate Database

```bash
bun run db:push
```

Seeding the Database

```bash
bun run db:seed
```
